SmartNotes - AI Study Assistant | Kotlin, MVVM, Gemini AI

- Built intelligent note-taking app leveraging Google's Gemini AI for 
  automatic note summarization and content enhancement
- Implemented robust MVVM architecture with Repository pattern, ensuring 
  separation of concerns and testable code
- Achieved 100% offline functionality through Room database with reactive 
  data flows using Kotlin Coroutines
- Delivered polished user experience with Material Design 3 components and 
  intuitive gesture-based interactions
